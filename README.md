# Yixuan Deng Academic Homepage

Static academic homepage for GitHub Pages.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## GitHub Pages Deployment

This site is intended for the repository:

```text
LanYin-coder/LanYin-coder.github.io
```

Push the `main` branch to GitHub. GitHub Pages will serve it at:

```text
https://LanYin-coder.github.io
```
